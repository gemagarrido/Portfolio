# Plan 3 — Página 404 y accesibilidad

## Fase 1: Prompt a la IA

¿Cómo puedo crear una página 404 personalizada coherente con la estética del portfolio y cumplir buenas prácticas de accesibilidad web?

## Fase 1: Respuesta de la IA (plan)

La IA propuso:

- Crear una página 404 con identidad visual coherente
- Separar HTML y CSS
- Incluir enlaces de navegación claros
- Implementar prefers-reduced-motion y skip link

## Fase 2: Implementación

Se implementó:

- Página 404 personalizada con HTML y CSS separados
- Enlace discreto a la 404 desde el footer (“Lost a gem?”)
- Mejores prácticas de accesibilidad (skip link, reduce motion)
