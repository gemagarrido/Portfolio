# Plan 2 — Animaciones con GSAP

## Fase 1: Prompt a la IA

¿Qué animaciones sencillas puedo implementar con GSAP y ScrollTrigger en un portfolio web para cumplir un nivel intermedio sin sobrecargar la experiencia?

## Fase 1: Respuesta de la IA (plan)

La IA sugirió:

- Animaciones de entrada (fade + desplazamiento vertical)
- Uso de ScrollTrigger para activar animaciones al hacer scroll
- Respetar prefers-reduced-motion para accesibilidad

## Fase 2: Implementación

Se implementaron:

- Animaciones de entrada en secciones clave (About, Proyectos, Footer)
- ScrollTrigger con valores suaves
- Desactivación automática de animaciones si el usuario tiene activado prefers-reduced-motion
