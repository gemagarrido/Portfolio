# Plan 1 — Navegación responsive

## Fase 1: Prompt a la IA

¿Cómo puedo implementar una navegación responsive con menú hamburguesa utilizando Bootstrap 5, manteniendo una estética editorial y minimalista?

## Fase 1: Respuesta de la IA (plan)

La IA propuso:

- Utilizar el componente Navbar de Bootstrap 5
- Implementar un menú hamburguesa para pantallas pequeñas
- Cerrar automáticamente el menú al hacer clic en un enlace
- Personalizar los estilos para mantener coherencia visual

## Fase 2: Implementación

Se implementó:

- Navbar con Bootstrap 5
- Menú hamburguesa funcional en móvil
- Cierre automático del menú al seleccionar una sección

Se adaptaron los estilos manualmente para ajustarlos a la identidad visual del portfolio.
